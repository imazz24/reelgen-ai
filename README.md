# 🎬 ReelGen AI

**AI-Powered Short-Form Video Concept Generator**

ReelGen AI helps content creators, marketers, and media teams generate professional reel concepts, scene breakdowns, captions, and hashtags in seconds.

---

## 🚀 Live Demo
**https://reelgen-ai.vercel.app**

---

## 💡 Features
- ✨ AI-style prompt-based workflow
- 🎬 Instant reel scene breakdown (Hook → Problem → Solution → Demo → CTA)
- 📝 Auto-generated captions and hashtags
- 🎨 Modern, media-industry-ready UI
- 📱 Fully responsive design
- ⚡ Lightning-fast generation

---

## 🛠️ Tech Stack
- **Frontend:** Next.js 14 (App Router), TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment:** Vercel
